# Ningyi Skills

Claude Code 自定义技能集合，存放于 `~/.claude/skills/` 并通过此仓库同步到云端。

## Skills 列表

| Skill | 描述 |
|-------|------|
| github-kb | GitHub 知识库管理 |
| humanizer | 去除 AI 写作痕迹 |
| ml-paper-writing | ML/AI 论文写作 |
| paper-writing-tips | 学术论文写作指南 |
| pdf | PDF 处理（提取/合并/拆分/OCR等） |
| scientific-story | 科研叙事构建（基金/论文） |
| skill-creator | Skill 创建指南 |

## 同步方式

```bash
# 将 ~/.claude/skills/ 同步到本仓库并推送到 GitHub
skill-sync
```
